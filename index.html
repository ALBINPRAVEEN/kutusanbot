Commands

Command Aliases Usage Description
bassboost bb, bass Enables bass boosting audio effect
clear cl, cls None Clears the server queue
config conf None Edit the bot settings
disconnect leave, exit, quit, dc, stop None Stop the music and leave the voice channel
grab save None Saves the current song to your Direct Messages
help command, commands, cmd [command] Information about the bot
invite inv None To invite me to your server
loop l, repeat None Loop the current song
loopqueue lq, repeatqueue, rq None Loop the whole queue
lyrics ly [Song Name] Shows the lyrics of the song searched
nowplaying np, nowplaying, now playing None See what song is currently playing
pause None Pauses the music
play p [song] Play your favorite songs
queue q None Shows all currently enqueued songs
remove rm [number] Remove a song from the queue
resume None Resumes the music
search se [song] Shows a result of songs based on the search query
seek forward Seek to a position in the song
shuffle shuff None Shuffles the queue
skip s, next None Skip the current song
skipto st Skip to a song in the queue
stats about, ping, info None Get information about the bot
volume vol, v Check or change the current volume
youtube yt None Starts a YouTube Together session
